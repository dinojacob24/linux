curl Command

Network Transfer

curl is a tool for transferring data from or to a server using various protocols including HTTP, HTTPS, FTP, FTPS, SCP, SFTP, and more.

Basic Syntax

curl [options] [URL]

Common Options

  • -o - Write output to file
  • -O - Write output to file with remote name
  • -X - Specify request method
  • -H - Add header to request
  • -d - Send POST data

Examples

# Download a file
curl -O https://example.com/file.zip

# Send POST request
curl -X POST -d "data=value" https://api.example.com

# Include headers
curl -H "Authorization: Bearer token" https://api.example.com

# Follow redirects
curl -L https://example.com

Pro Tips

  • Use -v for verbose output
  • Use -k to allow insecure connections
  • Use --progress-bar for download progress
