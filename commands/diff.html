diff Command

File Comparison

The diff command compares files line by line and shows the differences between them. It's commonly used to compare text files and create patches.

Basic Syntax

diff [options] file1 file2

Common Options

  • -u - Unified format output
  • -r - Recursively compare directories
  • -i - Ignore case differences
  • -b - Ignore changes in whitespace

Examples

# Compare two files
diff file1.txt file2.txt

# Compare with unified format
diff -u file1.txt file2.txt

# Compare directories
diff -r dir1/ dir2/

# Create a patch file
diff -u original.txt modified.txt > changes.patch

Pro Tips

  • Use colordiff for colored output
  • Use -w to ignore all whitespace
  • Use vimdiff for interactive comparison
