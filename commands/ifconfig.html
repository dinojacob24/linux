ifconfig Command

Network Configuration

ifconfig (interface configuration) is used to configure and display information about network interfaces on the system.

Basic Syntax

ifconfig [interface] [options]

Common Options

  • up - Activate interface
  • down - Deactivate interface
  • netmask - Set network mask
  • broadcast - Set broadcast address

Examples

# Display all interfaces
ifconfig

# Configure IP address
ifconfig eth0 192.168.1.100

# Enable interface
ifconfig eth0 up

# Disable interface
ifconfig eth0 down

# Set netmask and broadcast
ifconfig eth0 192.168.1.100 netmask 255.255.255.0 broadcast 192.168.1.255

Pro Tips

  • Use -a to show all interfaces, even if down
  • Modern systems prefer ip command over ifconfig
  • Check both IPv4 and IPv6 configurations
