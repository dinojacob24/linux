kill Command

Process Management

The kill command sends signals to processes. By default, it sends the TERM signal to terminate a process, but it can send various other signals for different purposes.

Basic Syntax

kill [options] pid

Common Signals

  • SIGTERM (15) - Graceful termination
  • SIGKILL (9) - Immediate termination
  • SIGHUP (1) - Hangup
  • SIGSTOP (19) - Stop process

Examples

# Basic termination
kill 1234

# Force kill process
kill -9 1234

# Kill by process name (using pkill)
pkill firefox

# List all signals
kill -l

Pro Tips

  • Use SIGTERM first before SIGKILL
  • killall can terminate multiple processes
  • Use ps or top to find PIDs
