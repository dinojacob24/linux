ping Command

Network Diagnostic

The ping command sends ICMP ECHO_REQUEST packets to network hosts to test connectivity and measure response time.

Basic Syntax

ping [options] destination

Common Options

  • -c count - Stop after sending count packets
  • -i interval - Wait interval seconds between packets
  • -s size - Specify packet size in bytes
  • -W timeout - Time to wait for response

Examples

# Basic ping
ping google.com

# Send specific number of packets
ping -c 4 192.168.1.1

# Change packet interval
ping -i 2 google.com

# Set packet size
ping -s 100 google.com

# Set timeout
ping -W 2 192.168.1.1

Pro Tips

  • Use Ctrl+C to stop continuous ping
  • Check both domain names and IP addresses
  • Use -q for quiet output (summary only)
