ssh Command

Remote Access

SSH (Secure Shell) is a protocol for securely connecting to remote systems. It provides encrypted communications between two untrusted hosts over an insecure network.

Basic Syntax

ssh [options] [user@]hostname [command]

Common Options

  • -p PORT - Connect to specific port
  • -i KEY - Use specific identity file
  • -X - Enable X11 forwarding
  • -v - Verbose mode

Examples

# Basic connection
ssh username@hostname

# Connect to specific port
ssh -p 2222 username@hostname

# Use identity file
ssh -i ~/.ssh/mykey username@hostname

# Execute remote command
ssh username@hostname 'ls -la'

Pro Tips

  • Use SSH keys instead of passwords for better security
  • Configure ~/.ssh/config for easier connections
  • Use agent forwarding for convenient key management
