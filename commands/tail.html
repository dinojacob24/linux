tail Command

File Viewing

The tail command outputs the last part of files. It's particularly useful for viewing log files and monitoring file changes in real-time.

Basic Syntax

tail [options] [file]

Common Options

  • -n N - Output last N lines
  • -f - Follow file changes
  • -F - Follow and retry if file inaccessible
  • -c N - Output last N bytes

Examples

# Show last 10 lines
tail file.txt

# Show last 20 lines
tail -n 20 file.txt

# Monitor file in real-time
tail -f log.txt

# Show last 100 bytes
tail -c 100 file.txt

Pro Tips

  • Use -f for log monitoring
  • Combine with grep for filtering
  • Use multiple files simultaneously
