top Command

System Monitoring

The top command provides a dynamic real-time view of a running system. It displays system summary information and a list of processes or threads currently managed by the Linux kernel.

Basic Syntax

top [options]

Common Options

  • -b - Batch mode operation
  • -n NUM - Number of iterations
  • -d SEC - Screen update interval
  • -p PID - Monitor specific process ID
  • -u USER - Display specific user processes

Interactive Commands

  • h or ? - Help screen
  • q - Quit top
  • k - Kill a process
  • r - Renice a process
  • f - Field management
  • 1 - Toggle CPU cores view

Examples

# Basic usage
top

# Update every 2 seconds
top -d 2

# Show only processes of user 'john'
top -u john

# Monitor specific process ID
top -p 1234

Pro Tips

  • Use M to sort by memory usage
  • Use P to sort by CPU usage
  • Use c to show command line path
  • Use W to save current settings
